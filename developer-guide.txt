Local setup:

$ git clone https://github.com/Jon-LaFlamme/MovieBuff.git

$ cd moviebuff

$ pip install -e .

$ export FLASK_APP=flaskr

$ export FLASK_ENV=development

$ flask run

HINT: "Cmd" + "Shift" + "r" for browser cache reset
